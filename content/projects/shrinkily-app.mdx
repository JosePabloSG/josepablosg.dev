Traditional URL shorteners lack flexibility and customization, offering only generic link shortening without advanced features like custom redirects, link protection, or performance tracking. This limits the needs of content creators and businesses seeking more control.

Shrinkily app lets users create personalized landing pages for each link with click tracking and link protection features. The platform provides an intuitive interface for managing shortened URLs while maintaining security through OAuth authentication.

The platform gives users full control over their links, enhancing marketing and content strategies. The combination of click tracking and customization options helps users better understand their link performance.

This project deepened my expertise in scalable architecture, security implementation, and performance optimization. Key achievements include implementing a custom redirect system supporting complex routing patterns and integrating OAuth for secure user authentication.
